# COLLEGE_TRACKER
An Android Application to track attendance of students and manage all needs of educational institutions. The app has plenty other features like attendance and note sharing, as well as mentoring of students through chat. Discussion forums for doubt discussion. Notice generation for students and Teachers. Clubs and Societies for students.
